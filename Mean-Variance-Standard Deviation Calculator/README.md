# Mean-Variance-Standard Deviation Calculator

This project calculates the mean, variance, standard deviation, max, min, and sum of a 3x3 matrix created from a list of 9 numbers.

## Files
- `main.py` - Entry point for the program
- `mean_var_std.py` - Calculator module
- `test_module.py` - Test module

## Requirements
See `requirements.txt` for dependencies.

## Running the Program
```bash
python main.py
```

## Usage
Pass a list of 9 numbers to the `calculate()` function to compute statistics.
