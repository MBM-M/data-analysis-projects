# Data Analysis with Python - FreeCodeCamp Projects

## Overview
A collection of data analysis projects completed as part of the FreeCodeCamp Data Analysis with Python curriculum. These projects demonstrate practical applications of data manipulation, visualization, and statistical analysis using Python.

## Projects

### Project 1: Mean-Variance-Standard Deviation Calculator
Calculate statistical measures from numeric data arrays.

### Project 2: Demographic Data Analyzer
Analyze demographic census data and provide demographic statistics.

### Project 3: Medical Data Visualizer
Visualize medical examination data with correlation matrices and categorical plots.

### Project 4: Page View Time Series Visualizer
Analyze and visualize time series data for website page views.

### Project 5: Sea Level Predictor
Predict future sea levels based on historical data using regression analysis.

## Technologies Used
- **Python 3**
- **Pandas** - Data manipulation and analysis
- **NumPy** - Numerical computing
- **Matplotlib** - Data visualization
- **Seaborn** - Statistical data visualization

## Getting Started

### Prerequisites
```bash
pip install pandas numpy matplotlib seaborn
```

### Running Projects
```bash
python project_name.py
```
