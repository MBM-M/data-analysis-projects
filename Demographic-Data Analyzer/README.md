# Demographic Data Analyzer

This project analyzes demographic data from the Adult dataset to compute various statistics about race distribution, education levels, salary distributions, and more.

## Files
- `main.py` - Entry point for the program
- `demographic_data_analyzer.py` - Main analysis module
- `adult.data.csv` - Dataset file

## Requirements
See `requirements.txt` for dependencies.

## Running the Program
```bash
python main.py
```
